# RTA介绍

## 名称由来

RTA诞生于2019年。程序化产研团队讨论如何为即将推出的新业务能力命名，其间列举了各种备用名称，既有沿袭RealTime命名的 `RT*` 等选项，也有完全抛弃RealTime概念的命名思路。考虑到功能形态有一定的RTB基因，且为了更便于市场接受，决定选用 `RT*` 命名方式。后面一个字母的选用并没有太多争论，主要是 `RTC` `RTX` 等热门选项在厂内已经有其他场景在用。为了避免冲突，最后决定使用 `RTA`为新产品命名。

## 能力发展

初期推出的RTA，通过以引入广告主一方数据，提升人群圈选准确率，在拉新拉活场景中取得了显著的效果。后期陆续推出了调价调权、实验系统、手机位图、联邦交互、商品推荐、商户推荐、广告推荐等高阶应用，服务于行业细分场景。时至今日，RTA已经承载了较大规模的广告业务消耗，成为互联网广告行业不可或缺的组成部分。
