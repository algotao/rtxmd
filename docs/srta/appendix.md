# 附录：行政区划码映射表

本文档提供中国省、直辖市、自治区及其下辖城市的行政区划码映射表。

:::info[说明]
- 行政区划码采用 6 位数字编码
- 省级行政区划码末位为 0000（如北京市：110000）
- 地级市行政区划码末位为 00（如石家庄市：130100）
- 特殊区域和县级市可能使用完整 6 位编码
:::

## 使用说明

### 数据读写

1. **数据空间**：与did的读写类似，常住地的读写ds为geofac。字段格式也一致。请先使用info命令查询当前账号是否开通了 `geofac` 数据区。
2. **用户ID**：用户ID使用地域编码，或是 `省:市` 的格式。例如 `130100` 或 `河北省:石家庄市`。

### lua内部查询规则

1. **精确匹配**：优先使用当前行政区域码进行查询。
2. **省级回退**：如果找不到数据，会自动回退到省级代码。
3. **国家级回退**：如果省份也找不到，最终回退到国家代码 `100000`
4. **有效数据**：只要该地域编码被写入过，及时内容全为0，也会被视为有数据。

代码示例

```lua
geofacData = srta.get_dsdata(srta.DS_GEOFAC) -- 获取常住地数据
```

### 特殊说明

- **直辖市**：直辖市的省份名称和城市名称相同（如北京市、上海市等）
- **自治州/盟**：使用完整的自治州或盟名称（如"延边朝鲜族自治州"、"锡林郭勒盟"）
- **县级市**：部分县级市有独立的行政区划码（如海南省的五指山市：469001）
- **特殊区域**：雄安新区等特殊区域也有独立的行政区划码（133100）
